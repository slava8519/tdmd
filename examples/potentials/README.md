# Open Potentials Library

This directory stores open interatomic potential files used by TDMD examples.

- Format currently bundled for runtime use: `eam/alloy` (DYNAMO setfl).
- Source catalogs:
  - official LAMMPS `potentials/` directory,
  - NIST Interatomic Potentials Repository (IPR) downloadable files.
- References:
  - `https://github.com/lammps/lammps/tree/stable/potentials`
  - `https://raw.githubusercontent.com/lammps/lammps/stable/potentials/README`
  - `https://www.ctcms.nist.gov/potentials/`

For bundled EAM files and provenance details, see:
- `examples/potentials/eam_alloy/README.md`
- `examples/potentials/eam_alloy/library.json`
- `examples/potentials/eam_alloy/SHA256SUMS`
